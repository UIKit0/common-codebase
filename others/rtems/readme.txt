
Follow below steps to create rtems-4.9.4 build envirement:

*) install ./MinGW/MinGW-5.1.6.exe, please install to d:/compiler/mingw;
    if install fail, please use SVN to checkout from my privite SVN address:
      https://cli-for-avr.googlecode.com/svn/trunk/MinGW

*) install ./MinGW/MSYS-1.0.11.exe, please install to d:/compiler/msys;

*) install ./MinGW/msysDTK-1.0.1.exe, please install to d:/compiler/msys;

*) install ./rtems4.9-tools-4.exe, please install to d:/compiler/rtems-4.9;
    if it popup for MinGW not installed, ignore it, and continue to install.

*) double click build.bat;

that's all.
